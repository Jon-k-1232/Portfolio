El Burrito Oso

This app allows a user to locate, read, and leave reviews for burritos. By inputting an address a user will be able to view restaurants in the area that sell burritos. A user will then see a rating for each hit on a scale of 1-10. The reason I thought to create this application was in order to find the best burritos in any given location. The best way to find top rated food was to crowed source the data.

TLX

This app was created to give tenants and property managers a single place to exchange. Specifically, tenants and property managers can converse through messages, as well as manage billing. I thought to create this app when I realized some property managers were still using paper invoices, and emails to converse. The issue with paper invoices and emails is both can be deleted, misplaced, or not delivered. This application keeps history of all messages and invoices to help maintain records and communication.
